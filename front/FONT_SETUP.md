# 폰트 설치 가이드

Clean Pro 디자인 시스템에서 사용할 폰트 2가지를 설치해야 합니다.

## 1. 카페24 써라운드 (Cafe24Ssurround)

**용도:** 큰 제목 (Display)

### 다운로드
https://noonnu.cc/font_page/363

또는

https://github.com/openhiun/cafe24

### 필요한 파일
- `Cafe24Ssurround.ttf` (또는 .otf)
- `Cafe24SsurroundAir.ttf` (옵션, 얇은 버전)

### 설치 위치
```
front/assets/fonts/Cafe24Ssurround.ttf
front/assets/fonts/Cafe24SsurroundAir.ttf
```

---

## 2. SUIT

**용도:** 본문, 중간 제목

### 다운로드
https://noonnu.cc/font_page/885

또는

https://github.com/sunn-us/SUIT/releases

### 필요한 파일
- `SUIT-Regular.ttf`
- `SUIT-Medium.ttf`
- `SUIT-SemiBold.ttf`
- `SUIT-Bold.ttf`

### 설치 위치
```
front/assets/fonts/SUIT-Regular.ttf
front/assets/fonts/SUIT-Medium.ttf
front/assets/fonts/SUIT-SemiBold.ttf
front/assets/fonts/SUIT-Bold.ttf
```

---

3. 앱 재시작
